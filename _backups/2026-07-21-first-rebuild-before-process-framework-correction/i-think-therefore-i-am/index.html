我思う故に我在り

I THINK, THEREFORE I AM / 我思故我在

制作年2024.11 媒体総合材料 形式インスタレーション／パフォーマンス

PROPOSITION 01

私が現実を観察しているのでしょうか。それとも、現実が私をつくっているのでしょうか。

現実と存在は固定されたものではなく、思考の投影と物理的な対象が重なり合うたびに再構築されます。本作でいう「リアル」は、物理的実体だけでなく、仮想空間のデジタルな存在や個人の記憶に残る像も含みます。「存在」は物質から思考まで、過去から未来までを横断します。

私は創作者であると同時に観察者となり、家や車、そして自分自身の像を、インスタレーションと行為によって組み替えます。鑑賞者には、「現実」と「存在」が時間と空間の中で変化する様子を、考えるだけでなく体験してほしいと考えています。

INSPIRATION

古いSF映画の未来は、いまを生きる私たちの現在であるはずなのに、現実はまだその姿になっていません。映画の中の未来は、当時の理性と感情がつくった、夢の中の並行世界のように見えました。

02

非線形研究星図

THOUGHT ↔ REALITY ↔ MEMORY

想像は未来を予測するのではなく、現在を映す。

ひとつの意識、複数の像。

CORE QUESTION

意識の投影と現実の存在のあいだには、何が生まれるのでしょうか。

確率と非決定論。想像と重なりが、私たちの知覚する状態を決めていきます。

SCIENCE FICTION

未来都市は、その時代の欲望と不安を映す

情報技術、オートメーション、電化への期待をたどると、日本の技術崇拝と技術不安が、同じ映画の中に共存していました。

REALITY

現実は重なりによって構築される

意識の投影はひとつでも、異なる複数の像として現れます。物理世界と意識の世界のあいだにも、ひとつの「実在」が立ち上がります。

COLLECTIVE COGNITION

存在は、複数のリアルの重なり

私たちが存在と呼ぶものは、個人の知覚だけでなく、集団の認知によって形成される重層的なリアルです。

MEMORY / OLD HOUSE

恐ろしかった古い家は、静かな廃墟になった

成長とともに変わったのは建物でしょうか、私の認識でしょうか。その記憶から、失われた家を実験的に再構成したいと考えました。

INTERVIEW / 20s—40s

20s

SFが現実になると、映画の美しさと現実の間には差が現れます。

40s

03

画像再構成の系譜

EXTRACT → FRACTURE → OVERLAP

IMAGE RESEARCH

日常の像を分解し、存在しない存在を組み立てる

SF映画に現れる多くの道具は、日常生活から生まれています。車と家を、機能ごとの断片へ分け、異なる時代の像や記憶と重ね合わせました。

記憶の中の旧宅を再構築するとき、私は失われた建物を再現するだけでなく、「存在しない存在」をつくっています。もし意識が現実を再構築できるなら、そこに立つ私は何なのでしょうか。

04

比較実験室：光・空気・水

INTANGIBLE ↔ TANGIBLE

MEDIUM TEST

像を「実在する物体」から離し、虚と実のあいだに置くために、性質の異なる三つの媒体を同じ問いのもとで比較しました。

光

プロジェクターで光と影をつくり、触れることのできない像を空間に現します。異なる照明環境で、像が物体から離れる状態を観察しました。

空気

目に見えない空気は、風船によって一時的な形を得ます。その表面へ像を重ねることで、有形と無形が同じ場所に共存します。

水

水は透明でありながら、触れることのできる実在です。撮影後の画像を水に溶かし、水の中で像を分離し、再構成しました。

05

選定した方法

AIR / BALLOON / COLLAGE

SELECTED MEDIUM

FROM THE INTANGIBLE TO A TEMPORARY BODY

空気／風船

空気は目に見えませんが、風船の膜によって一時的な輪郭を持ちます。その不安定な身体に、家、車、記憶、そして私自身の像をコラージュします。

像は表面に固定されているように見えても、内部の空気が失われれば形ごと崩れます。この関係を、複数のリアルが重なって成立する「存在」のモデルとして用いました。

01虚と実のあいだを、触れられる輪郭として示す

02異なる時代と記憶の像を、同じ表面に重ねる

03日常の物体を、不安定で変化する身体へ置き換える

06

シミュレーションと制作

MODEL → SCALE → SURFACE

3D SIMULATION

家と車を、同じ風景の中で組み直す

3Dモデルを用いて、再構成された家、車、人の尺度と配置を比較しました。ここでは最終的な技術仕様ではなく、物体同士の関係と見え方を検討しています。

FABRICATION

断片を、膨らむ表面へ

印刷した像を切り分け、風船で形づくられた家と車の表面へ重ねました。制作の途中でも、像の境界と物体の輪郭は絶えず変化します。

07

三段階のパフォーマンス

INHABIT → PIERCE → RELEASE

A BODY MOVES THROUGH THREE STATES

作品は、ひとつの身体が「重なる」「破れる」「手放される」という三つの状態を通過する時間として構成されています。

01 / INHABIT

風船に自己を重ねる

空気で膨らむ身体に再構成した像と私の肖像をまとわせ、私は風船へ入り込みます。風船は、もうひとつの私になります。

02 / PIERCE

投影された私を刺し破る

思考の投影としてつくられた「私」を刺し破ります。複数のリアルで保たれていた輪郭が崩れ、その確かさが虚構へ反転します。

03 / RELEASE

残された像を風に返す

二元論的な対立を問い直した後、残されたものを凧へ変え、風に乗せて手放します。「認識は現実なのか」という問いを、答えで閉じず空へ返します。

08

OBSERVER / OBJECT / FIELD

THE VIEWER ENTERS THE QUESTION

展示では、私は観察する主体であると同時に、観察される対象になります。鑑賞者の視線と行為が加わるたびに、作品は別の現実として読み替えられます。
